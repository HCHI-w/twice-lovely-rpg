TWICE Lovely RPG

A small RPG fan game featuring members of TWICE.
Built with Godot 4 and playable directly in the browser.
  
  
🎮 Play Online  
https://hchi-w.github.io/twice-lovely-rpg/
    
    
Game Overview
  
TWICE Lovely RPG is a turn-based RPG prototype where players can:  
。Choose 3 TWICE members as their team  
。Assign different classes to each character  
。Fight through multiple stages of enemies  
。Use skills and strategy to win battles  
  
The game is designed as a lightweight browser RPG and runs on desktop and mobile browsers.

  
Features  
。Character selection system  
。Job / class selection system  
。Turn-based battle system  
。Stage progression  
。Web playable (HTML5 export)  
  
Built With  
。Godot Engine 4.6  
。GDScript  
。HTML5 / Web export  
。Hosted via GitHub Pages  
  
  
🎮 How to Play  
1.Open the game page  
2.Click Start Game  
3.Choose 3 characters  
4.Select a class/job for each character  
5.Enter battle and defeat enemies to progress
